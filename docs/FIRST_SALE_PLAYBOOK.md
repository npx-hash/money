# First Sale Playbook (14 Days)

## Day 1-2
- Verify Stripe checkout + webhook in live mode.
- Seed products and QA each PDP/cart/checkout path.
- Publish trust assets: shipping policy, refund policy, contact email.

## Day 3-5
- Post 2-3 UGC-style short videos/day for each product.
- Build one collection-focused landing ad angle per product.
- Capture emails with a 10% first-order incentive.

## Day 6-8
- Launch $10/day retargeting campaign to cart visitors.
- Keep prospecting organic; only retarget paid early.

## Day 9-11
- Run safe price test variants (admin pricing panel).
- Pause underperforming SKU via kill-switch.

## Day 12-14
- Double down on best converting SKU.
- Add urgency bundle copy and social proof snippets.
- Aim for 3-5 sales/day at protected margin.

## Minimum economics target
- Unit profit target: $5-$10 net after payment fees/ad spend.
- Gross margin floor: 30% enforced by platform rules.
